\doxysubsection{Detailed Description}
EEPROM control logic for 28Cx EEPROM programmer. 

Contains functions for setting up and controlling the EEPROM itself. This includes pin initialization and any low-\/level control functions needed to interface with the EEPROM.

enum \mbox{\hyperlink{eeprom_8h_aa45819052ed50f125c644ff22066b7d6}{ROMMode}}}



Enumeration for the different modes the EEPROM can be in (READ, WRITE, STANDBY) 

This enum is used to track the current mode of the EEPROM and to configure the control lines and data bus pins accordingly when changing modes. The modes are\+:
\begin{DoxyItemize}
\item STANDBY\+: The EEPROM is in a low-\/power standby state, with control lines configured to disable both reading and writing.
\item READ\+: The EEPROM is configured for reading, with control lines set to enable reading and data bus pins set as inputs.
\item WRITE\+: The EEPROM is configured for writing, with control lines set to enable writing and data bus pins set as outputs. 
\end{DoxyItemize}
