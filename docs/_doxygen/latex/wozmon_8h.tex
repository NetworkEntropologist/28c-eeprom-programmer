\doxysubsection{Detailed Description}
Header file for WOZMON monitor code and related functions. 

This file contains the declaration of the \doxylink{wozmon_8h_ab64f1beee79bed2a4dbfca110a0febd9}{write\+Woz\+Mon()} function, which is responsible for writing the WOZMON monitor code to the EEPROM. It also defines the starting address for the WOZMON code in the EEPROM.
